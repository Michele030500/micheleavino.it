Web portal for MASE dedicated to the optimized and intuitive management of documentation, offering users a centralized environment for accessing and quickly searching information. The portal was designed with a user-friendly interface and a categorized structure that facilitates navigation and retrieval of resources, while ensuring data security through an authentication system.
In parallel, we built a custom application using Power BI to dynamically visualize and analyze key performance indicators (KPIs) extracted from machine learning models
